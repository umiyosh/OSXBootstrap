# OSXBootstrap

# a precondition

* install xcode from AppStore. And Command line tools.
* <code>ssh-kegen -t rsa</code> and add setting to Deploy key

# instruction

~~~
% git clone https://github.com/umiyosh/OSXBootstrap.git
% cd OSXBootstrap/
% ./bootstrap.sh
~~~
